This is application for parents. Run it and it will shut down computer after some time, so your kid have life becides computer games.
Just run it, and this application will shutdown Windows computer in 1 hour. 

You may use parameter in minutes, to set different time span:
For example to turn off after 90 min:

c:\..\ParentalShutdown.exe 90

Recompiled it under .Net Framework 4.7.2 for x64 bit Windows 10 PC.
