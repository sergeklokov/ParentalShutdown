This is free application for parents. 
Free parental control over time that computer will run. No ads of course.
Run it and it will shut down computer after some time, so your kid have life becides computer games.
Just run it, and this application will shutdown Windows computer in 1 hour. 

You may use parameter in minutes, to set different time span:
For example to turn off after 90 min:

c:\..\ParentalShutdown.exe 90

Use shortcuts files to understand how to set parameter in minutes.
Download app from the .zip file and copy at any folder on your PC.
